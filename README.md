# Social Media Agent using CrewAI

An intelligent AI agent built with CrewAI for automated social media content creation, scheduling, and engagement.

## Features
- Automated content generation
- Multi-platform posting
- Engagement analytics
- AI-powered content optimization

## Tech Stack
- CrewAI
- Python
- OpenAI API

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
